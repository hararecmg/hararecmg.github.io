Juego de fuentes tipográficas

Las siguientes listas de fuentes tipográficas se complementan entre sí, además están ordenadas poniendo primero la fuente principal, después la fuente secundaria, y al último la fuente complementaria. La fuente principal se utiliza para el título principal y los encabezados, la fuente secundaria se utiliza para el cuerpo del texto y los subtítulos, y la fuente complementaria se utiliza para cualquier elemento gráfico o elemento de diseño, como tablas o infografías.
